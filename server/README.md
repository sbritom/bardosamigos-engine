# Bar Radio Engine Server

Backend Node interno da Sprint 2.

## Rodar localmente

```bash
node server/src/index.js
```

## Variaveis

- `RADIO_STREAM_URL`
- `RADIO_STATUS_URL`
- `RADIO_ADMIN_USER`
- `RADIO_ADMIN_PASSWORD`
- `AUTO_DJ_ENABLED`
- `RADIO_PROVIDER`
- `RADIO_SERVER_PORT`

Upload, Supabase, Storage, AutoDJ real, Icecast real e streaming real ficam para a Sprint 3.
